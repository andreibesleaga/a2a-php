üöÄ Overview & Features

The A2A PHP SDK is a comprehensive implementation of the A2A (AI Agent-to-Agent) Protocol v0.3.0, providing a robust foundation for building AI agent communication systems in PHP.

üèÜ Key Features

  • Full A2A Protocol Compliance - 100% compliant with A2A Protocol v0.3.0
  • Modern PHP 8.0+ - Leverages strict typing, enums, and modern patterns
  • PSR Compliance - PSR-4 autoloading, PSR-3 logging
  • Comprehensive Testing - Unit, integration, and end-to-end tests
  • HTTP Client Abstraction - Flexible HTTP client with Guzzle
  • Real-time Communication - Server-Sent Events (SSE) streaming
  • Task Management - Complete task lifecycle management
  • Push Notifications - WebSockets and HTTP-based notifications
  • Error Handling - Comprehensive exception hierarchy
  • HTTPS/TLS Support - Production-grade security

üîß Architecture

The SDK is organized into several key components:

  • Core Protocol - A2AClient, A2AServer, A2AProtocol_v0_3_0 classes
  • Models - Data structures for agents, messages, tasks, and artifacts
  • Streaming - Real-time communication via SSE
  • Task Management - Complete task lifecycle and state management
  • Utilities - HTTP client, JSON-RPC, and helper functions

This implementation has been thoroughly tested against the official A2A Test Compatibility Kit:

  • Mandatory Features (25/25) - JSON-RPC 2.0, Agent Cards, Message Send, Task Management
  • Capability Features (14/14) - Message Capabilities, Push Notifications, Streaming

‚ö° Installation & Quick Start

Install via Composer

composer require andreibesleaga/a2a-php

Basic Usage

use A2A\A2AClient;
use A2A\Models\v0_3_0\AgentCard;
use A2A\Models\v0_3_0\Message;
use A2A\Models\AgentCapabilities;

// Create agent card
$capabilities = new AgentCapabilities(['message/send'], ['task/get', 'task/cancel']);
$agentCard = new AgentCard('MyAgent', 'Description', 'http://localhost', '1.0.0', $capabilities, [], []);

// Create client and send message
$client = new A2AClient($agentCard);
$message = new Message('msg-1', 'sender', 'Hello World!');
$response = $client->sendMessage('http://other-agent.com', $message);

Verify Installation

# Start the complete server (recommended)
php -S localhost:8081 examples/complete_a2a_server.php

# Test agent communication
php examples/complete_agent_communication.php

# Check implementation status
php examples/implementation_status.php

üíª Usage Examples

Basic Agent Setup

use A2A\Models\v0_3_0\AgentCard;
use A2A\Models\AgentCapabilities;
use A2A\A2AProtocol_v0_3_0;
use A2A\A2AServer;

// Create agent capabilities
$capabilities = new AgentCapabilities(
    ['message/send', 'task/create'],
    ['task/get', 'task/cancel', 'agent/card/get']
);

// Create agent card
$agentCard = new AgentCard(
    'MyAI', 
    'AI Assistant Agent', 
    'https://my-agent.com', 
    '1.0.0',
    $capabilities,
    ['text', 'json'],
    ['text', 'json']
);

// Initialize protocol and server
$protocol = new A2AProtocol_v0_3_0($agentCard);
$server = new A2AServer($protocol);

Sending Messages

use A2A\A2AClient;
use A2A\Models\v0_3_0\Message;
use A2A\Models\TextPart;

$client = new A2AClient($agentCard);

// Create a simple text message
$textPart = new TextPart('Hello, how can I help you?');
$message = new Message('msg-123', 'MyAI', [$textPart]);

// Send to another agent
$response = $client->sendMessage('https://other-agent.com', $message);

Task Management

use A2A\Models\v0_3_0\Task;
use A2A\Models\TaskStatus;

// Create a new task
$task = new Task('task-456', TaskStatus::PENDING);
$task->setInitialMessage($message);

// Get task status
$taskData = $client->getTask('task-456');

// Cancel task if needed
$client->cancelTask('task-456');

Push Notifications

use A2A\Models\PushNotificationConfig;
use A2A\Models\PushNotificationAuthenticationInfo;

// Configure webhook notifications
$authInfo = new PushNotificationAuthenticationInfo('bearer', 'your-token');
$config = new PushNotificationConfig(
    'http://your-webhook.com/notifications',
    $authInfo
);

$client->setPushNotificationConfig('task-456', $config);

Error Handling

use A2A\Exceptions\A2AException;
use A2A\Exceptions\TaskNotFoundException;

try {
    $task = $client->getTask('non-existent-task');
} catch (TaskNotFoundException $e) {
    // Handle task not found
    echo "Task not found: " . $e->getMessage();
} catch (A2AException $e) {
    // Handle other A2A errors
    echo "A2A Error: " . $e->getMessage();
}

Streaming Events

use A2A\Streaming\SSEStreamer;
use A2A\Client\StreamingClient;

// Server-side streaming
$streamer = new SSEStreamer();
$streamer->start('task-456');

// Client-side consumption  
$streamingClient = new StreamingClient($agentCard);
$streamingClient->resubscribeTask('task-456');
