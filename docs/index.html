Quick Start

Create an Agent

<?php
use A2A\A2AServer;
use A2A\Models\AgentCard;
use A2A\Models\AgentCapabilities;
use A2A\Models\AgentSkill;

$capabilities = new AgentCapabilities(
    canReceiveMessages: true,
    canSendMessages: true,
    canManageTasks: true
);

$skills = [
    new AgentSkill(
        'text_processing',
        'Process and analyze text content',
        ['text/plain']
    )
];

$agentCard = new AgentCard(
    name: 'My Agent',
    description: 'A helpful AI agent',
    url: 'https://my-agent.com',
    version: '1.0.0',
    capabilities: $capabilities,
    defaultInputModes: ['text/plain'],
    defaultOutputModes: ['text/plain'],
    skills: $skills
);

$server = new A2AServer($agentCard);
$server->start();

Connect to an Agent

<?php
use A2A\A2AClient;
use A2A\Models\Message;
use A2A\Models\TextPart;

$client = new A2AClient('https://agent.example.com');

// Get agent capabilities
$agentCard = $client->getAgentCard();
echo "Agent: " . $agentCard->getName();

// Send a message
$message = new Message(
    role: 'user',
    parts: [new TextPart('Hello, agent!')]
);

$response = $client->sendMessage($message);
echo $response->getParts()[0]->getText();

Task Management

<?php
use A2A\Models\Task;
use A2A\Models\Message;
use A2A\TaskManager;

$taskManager = new TaskManager();

// Create a task
$message = new Message('user', [new TextPart('Analyze this document')]);
$task = $taskManager->createTask($message);

// Monitor task progress
$status = $taskManager->getTask($task->getId());
echo "Task status: " . $status->getStatus()->value;

API Reference

Core Classes
  • A2AServer - Main server implementation
  • A2AClient - Client for connecting to agents
  • A2AProtocol_v0_3_0 - Protocol v0.3.0 handler
  • TaskManager - Task lifecycle management
  • AgentCard - Agent capability description
  • Message - Communication messages
  • Task - Asynchronous task representation
  • GrpcClient - High-performance gRPC client
  • StreamingClient - Real-time streaming client
  • Storage - Multi-driver persistence layer
Message Types
  • TextPart - Plain text content
  • FilePart - File attachments
  • DataPart - Structured data
  • FileWithBytes - Inline file data
  • FileWithUri - File references
  • SecurityScheme - Authentication schemes
  • AgentExtension - Protocol extensions
  • AgentInterface - Transport interfaces
  • PushNotificationConfig - Webhook configs
Protocol Methods
Method Description Parameters Returns
getAgentCard Retrieve agent capabilities None AgentCard
ping Health check None Pong response
sendMessage Send message to agent Message Message response
tasks/send Create asynchronous task Message Task ID
tasks/get Get task status Task ID Task status
